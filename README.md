# Buildit-Weather-App
> [Buildit-Weather-App](https://b-weather-app.herokuapp.com/)

# Uses

* React
* Redux
* React Router
* axios
* react-google-maps
* react-sparklines
* redux-promise

## Steps to Run application


**1.Install Dependencies**

	 npm install;

**2.Build Application**

	npm run postinstall;

**3.Run Application**

	 npm start //or
   npm run dev

**4.Browse the following url in Browser**

[Buildit-Weather-App](http://localhost:8080/)
